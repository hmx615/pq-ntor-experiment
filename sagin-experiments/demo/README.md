## 🎮 操作指南

### 基本操作
- **旋转地球**: 鼠标左键拖动
- **缩放视图**: 鼠标滚轮
- **查看节点信息**: 鼠标悬停在节点上
- **查看链路参数**: 鼠标悬停在链路上

### 切换拓扑
- 点击右上角的12个按钮
- 上半部分: 上行通信拓扑 (拓扑1-6)
- 下半部分: 下行通信拓扑 (拓扑7-12)

## 🎨 颜色说明

### 节点颜色
- 🟡 **黄色** = 卫星 (SAT)
- 🔵 **青色** = 无人机 (Aircraft)
- 🟢 **绿色** = 地面终端 (Ground)

### 链路颜色
- 🟢 **绿色实线** = 高RSSI (信道质量优秀)
- 🟠 **橙色实线** = 低RSSI (信道质量一般)
- 🟣 **紫色虚线** = 协作链路 (NOMA组内协作)

## 🐛 遇到问题？

### 页面打不开
- 检查是否有网络连接 (需要加载CDN库)
- 尝试使用Chrome或Firefox浏览器

### 地球不显示
- 检查浏览器是否支持WebGL
- 打开浏览器控制台查看错误信息

### 按钮无响应
- 刷新页面重试
- 查看是否有JavaScript错误
