## 🔑 核心概念

### NOMA (非正交多址接入)
- **原理**: 同一频段同时服务多个用户
- **功率分配**: 远端用户高功率，近端用户低功率
- **SIC**: 串行干扰消除技术
- **分组**: <近端, 远端> 配对

### RSSI (接收信号强度指示)
- **高RSSI**: 信道质量好，近端用户
- **低RSSI**: 信道质量差，远端用户
- **映射**: RSSI → 延迟/带宽/丢包率

### 上行 vs 下行
- **上行**: 终端→卫星，无组内协作
- **下行**: 卫星→终端，有组内协作
- **协作**: 近端帮助远端解码
