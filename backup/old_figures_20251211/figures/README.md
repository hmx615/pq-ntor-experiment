## 📊 核心数据速查

### 算法层面（无网络）
- PQ-NTOR握手: **30.71 μs** (快80.3%)
- Classic NTOR握手: 155.85 μs
- PQ消息大小: 1620字节（大14倍）

### 网络层面（12拓扑SAGIN）
- 电路建立时间差异: **0.00 ms** (完全相同)
- 端到端性能差异: **6.6%** (Classic慢)
- 测试成功率: **100%** (240/240)

### 关键发现
> 在90-240ms的SAGIN网络延迟下，0.125ms的握手差异完全被淹没，证明PQ-NTOR在卫星通信中性能无损。
